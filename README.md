# تطبيق قائمة المهام الذكية (Todo App)

تطبيق ويب بسيط وذكي لإدارة المهام اليومية، مصمم بواجهة مستخدم عصرية تدعم اللغة العربية (RTL).

## المميزات
- إضافة مهام جديدة مع تحديد الأولوية (عادية، مهمة، عاجلة).
- تتبع حالة المهام (مكتملة، قيد الانتظار).
- إحصائيات مباشرة لعدد المهام الكلي والمكتملة والمعلقة.
- تصفية المهام حسب الحالة.
- واجهة متجاوبة تعمل على جميع الأجهزة.

## التقنيات المستخدمة
- **HTML5**: لهيكلة محتوى التطبيق.
- **CSS3**: للتصميم والتنسيق (يدعم الوضع المتجاوب).
- **JavaScript (Vanilla)**: للتحكم في منطق التطبيق وتخزين البيانات.

## كيفية التشغيل
1. قم بتحميل ملفات المشروع.
2. افتح ملف `index.html` في أي متصفح ويب حديث.

## المتطلبات
لا يتطلب هذا المشروع أي خوادم أو قواعد بيانات خارجية، حيث يعمل بالكامل على المتصفح.
